# phytocommercefooter

Adds a small Phyto Commerce branding credit to the store footer.

## Hooks Registered

| Hook | Purpose |
|------|---------|
| `displayFooter` | Renders "Created with ❤️ from Phyto Commerce" credit |

## Configuration Keys

None.

## DB Tables

None — no custom tables required.

## Inter-module Dependencies

None. Standalone module.
