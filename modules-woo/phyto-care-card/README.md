# phyto-care-card

**Phyto Care Card for WooCommerce** — Generate downloadable PDF care guides per product. Optionally attach to order emails.

## Purpose

This plugin auto-generates a formatted PDF care guide for each WooCommerce product. An admin fills in care fields via a meta box on the product edit screen. Customers can download the PDF directly from the product page, and the PDF can optionally be attached to the WooCommerce order completion email.

## Installation

1. Upload the `phyto-care-card/` folder to `/wp-content/plugins/`.
2. Activate the plugin in **WP Admin → Plugins**.
3. WooCommerce 8.0 or later must be active.
4. After activation, go to **Settings → Permalinks** and click **Save Changes** to flush the rewrite rules that power the PDF download endpoint.

## Care Fields

Fill in these fields from the **Plant Care Card** meta box on any product edit page:

| Field Name | Meta Key | Type | Notes |
|---|---|---|---|
| Light Requirements | `_phyto_cc_light_req` | Select | Full Sun / Partial Shade / Bright Indirect / Low Light / Grow Light |
| Watering | `_phyto_cc_watering` | Text | Free text, e.g. "Water when top 2 cm dry" |
| Humidity | `_phyto_cc_humidity` | Text | e.g. "60–80% RH" |
| Temperature Min (°C) | `_phyto_cc_temp_min` | Number | Decimal allowed |
| Temperature Max (°C) | `_phyto_cc_temp_max` | Number | Decimal allowed |
| Potting Media | `_phyto_cc_potting_media` | Text | e.g. "Chunky bark + perlite 60/40" |
| Fertilisation | `_phyto_cc_fertilisation` | Text | e.g. "Half-strength balanced NPK fortnightly" |
| Dormancy Notes | `_phyto_cc_dormancy_notes` | Textarea | Full paragraph |
| Special Care Tips | `_phyto_cc_special_tips` | Textarea | Full paragraph |
| Attach to Order Email | `_phyto_cc_attach_email` | Checkbox (1/0) | See below |

The Download button only appears on the product page when at least one care field is non-empty.

## PDF Layout

The generated PDF is A4 portrait, built from raw PDF 1.4 syntax — no external library required.

- **Header bar** — forest-green (#1a3c2b) rectangle spanning full width with "Plant Care Guide" in white Helvetica-Bold 18pt and the product name in white Helvetica 13pt. "PhytoCommerce" appears in sage (#7dab8a) in the top-right corner.
- **Two-column grid** — Light, Water, Humidity (left column); Temperature, Potting Media, Fertilisation (right column). Each pair shows the field label in Helvetica-Bold 9pt followed by the value in Helvetica 9pt.
- **Full-width sections** — Dormancy Notes and Special Care Tips each appear below the grid with a section heading in Helvetica-Bold.
- **Footer** — "Generated by PhytoCommerce · phyto-evolution.github.io/PhytoCommerce" in small grey text.

## Download URL

```
https://yourstore.com/care-card/{product_id}/
```

The URL resolves via a custom WordPress rewrite rule. No server-level configuration needed. The PDF is served inline with `Content-Disposition: inline`.

## Email Attachment Setup

1. Open the product in WP Admin.
2. In the **Plant Care Card** meta box, check **Attach this care card PDF to the customer completed-order email**.
3. Save the product.

When a customer's order is marked Complete and any of the order's products have the checkbox ticked, the plugin generates the PDF and saves it to:

```
{wp-uploads}/phyto-care-cards/product-{id}.pdf
```

The file path is appended to the WooCommerce email attachment list and delivered with the order completion email.

## Hooks

### `phyto_cc_fields_output`

Filter the rendered care-fields HTML block before it is added to the page. Return an empty string to suppress. (Apply at `woocommerce_single_product_summary` priority 35.)

### `phyto_cc_attach_email`

Filter the array of file paths attached to the order completion email, after the plugin has appended its PDFs. Return the modified array.
