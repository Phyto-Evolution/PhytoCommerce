# PhytoCommerce

Open source ecommerce modules, themes and tools built for botanical and plant businesses. Part of the Phyto Evolution ecosystem.

## Modules
- **phytocommercefooter** — Replaces default PrestaShop branding with Phyto Commerce branding across any theme

## Requirements
- PrestaShop 8.1.x
- PHP 8.1+

## Installation
Clone this repo and copy the desired module folder into your PrestaShop `/modules` directory, then activate from the admin panel.
