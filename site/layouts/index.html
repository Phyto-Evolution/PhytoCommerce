What Is PhytoCommerce?

PhytoCommerce is a platform-agnostic e-commerce module suite built for plant businesses — tissue-culture labs, nurseries, rare-plant retailers, and botanical collectors. Every module solves a real operational gap that generic platforms ignore: tracking TC batch provenance, printing phytosanitary-aware invoices, gating prices behind KYC, or notifying customers when a seasonal species is back in stock.

The first implementations target PrestaShop 8 (28 modules, production-ready) and WooCommerce (actively porting). OpenCart, BigCommerce, and a bespoke scratch-built plant-commerce stack are on the roadmap — same feature set, idiomatic code for each platform.

If you run your own e-commerce server the source is open. If you want a fully managed store tuned for your catalogue, we handle the entire stack — reach out to discuss.

Quick Install (PrestaShop 8)

Upload phytocommerce_pack and click Install. All modules deploy in the correct dependency order. Or drop any single module into your /modules/ directory.

Quick Install (WooCommerce)

Upload the plugin ZIP via Plugins → Add New → Upload, or copy the plugin folder to wp-content/plugins/ and activate.
