What Is PhytoCommerce?

PhytoCommerce is a PrestaShop 8 module suite built specifically for specialty plant businesses — tissue-culture labs, nurseries, rare plant retailers, and botanical collectors. Each of the 28 modules addresses a real operational need that generic e-commerce platforms ignore.

From tracking TC batch provenance through to phytosanitary compliance, B2B wholesale tiering, customer plant journals, and live arrival guarantees — the suite covers the full lifecycle of a specialty plant sale.

Quick Install

Upload phytocommerce_pack to PrestaShop and click Install. All modules deploy in the correct dependency order. Or install any module individually by copying it to your /modules/ directory.

Source Code

All modules are open source and live at {{ .Site.Params.repoModules }} ↗. Each module directory contains its own README.md, SQL install/uninstall scripts, PHP classes, Smarty templates, and frontend assets.
